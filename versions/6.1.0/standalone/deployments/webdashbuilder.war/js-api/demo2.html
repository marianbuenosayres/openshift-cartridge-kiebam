How to embed table reports in your own application

This is an example of a dynamic sales report, that is embedded by using a simple javascript command:

dashbuilder_embed_table('chart1', 'http://localhost:8080/dashbuilder', 'kpi_29761353668431694', 800, 600, 1300, 1000);
